# Setup notes

2. Setting Up Your VS Code Environment

You don't need to write Python from scratch; you just need to install the environment and "fill in the blanks" using the examples from the Percepxion documentation.

Install Python: Ensure you have Python installed.

Install FastMCP and other libraries: Run this command in your VS Code terminal: 

pip install fastmcp requests python-dotenv 

Upload MCP file: Upload file named percepxion_mcp.py.

Add credentials to .env: Rename example.env to .env and add username and password.

Run the Server: In your terminal, run python percepxion_mcp.py

Update settings in Claude Desktop: upload claude_desktop_config.json file
